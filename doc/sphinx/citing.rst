##################
Acknowledging Yade
##################

We kindly ask YADE users to cite the documentation in scientific publications as a way to assess YADE's contribution to their field.

For the YADE project in general, please reference the documentation as a whole:
- \ V. Šmilauer et al. (2021), Yade Documentation 3rd ed. The Yade Project. `DOI:10.5281/zenodo.5705394 <https://dx.doi.org/10.5281/zenodo.5705394>`
For newer features of YADE, please reference the review article:
- \ V. Angelidakis et al. "YADE-An extensible framework for the interactive simulation of multiscale, multiphase, and multiphysics particulate systems". Computer Physics Communications 304 (2024): 109293. `DOI:10.1016/j.cpc.2024.109293 <https://doi.org/10.1016/j.cpc.2024.109293>`

Ideally, users cite both documents to acknowledge the work of both earlier and later developers of YADE. Beyond acknowledgment, proper referencing helps finding new use cases and new users by tracking the citations on `Yade's Scholar profile <https://scholar.google.com/citations?user=hZB8GGcAAAAJ&hl=en>`_.
